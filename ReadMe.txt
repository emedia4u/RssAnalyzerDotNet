Simple RSS Feed Analyzer
Determines if companie had no activity on RSS Feed for a given number of days.

Usage:
RssAnalyzerDotNet.exe companyName RssLink daysInactive
RssAnalyzerDotNet.exe Politico https://www.politico.com/rss/politicopicks.xml 2

Technologies:
The is .Net Framwork 4.72 console application

Latest code is in Development branch https://github.com/emedia4u/RssAnalyzerDotNet
Values hard coded to simplify debugging.
